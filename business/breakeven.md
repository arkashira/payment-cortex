# breakeven.md
## Unit Economics
### Cost per Active User
Based on the payment-cortex infrastructure, we estimate the following costs:
* Compute: $0.05 per user per month (assuming 10% utilization of a $5/month instance)
* Storage: $0.01 per user per month (assuming 1GB storage per user at $0.10/GB-month)
* Bandwidth: $0.005 per user per month (assuming 100MB/month at $0.05/MB)
Total cost per active user: $0.065 per month

## Pricing Tiers
We propose the following pricing tiers for payment-cortex:
### Tier 1: Starter
* Price: $9.99/month
* Features: 100 transactions/month, 1 user account, basic support
### Tier 2: Growth
* Price: $49.99/month
* Features: 1,000 transactions/month, 5 user accounts, priority support
### Tier 3: Enterprise
* Price: $199.99/month
* Features: 10,000 transactions/month, 20 user accounts, dedicated support

## Customer Acquisition Cost (CAC) Range
Based on industry benchmarks, we estimate the CAC range for payment-cortex to be between $50 and $200 per user.

## Lifetime Value (LTV) Estimate
Assuming an average user lifetime of 12 months and an average revenue per user (ARPU) of $29.99 (based on the pricing tiers), we estimate the LTV to be:
LTV = ARPU x user lifetime = $29.99 x 12 = $359.88

## Break-even Analysis
To calculate the break-even point, we need to determine the number of users required to cover the CAC and reach profitability.
Break-even users count = CAC / (LTV - CAC)
Assuming a CAC of $125 (midpoint of the estimated range), we get:
Break-even users count = $125 / ($359.88 - $125) ≈ 53 users

## Path to $10K MRR
To reach $10,000 in monthly recurring revenue (MRR), we can estimate the required number of users for each tier:
* Tier 1: $9.99/month → 1,001 users required to reach $10,000 MRR
* Tier 2: $49.99/month → 200 users required to reach $10,000 MRR
* Tier 3: $199.99/month → 50 users required to reach $10,000 MRR
A possible path to $10K MRR could be:
* 50 users on Tier 3 (Enterprise) = $9,999.50 MRR
* 10 users on Tier 2 (Growth) = $499.90 MRR
Total MRR: $10,499.40

This combination of users across different tiers can help us reach the $10,000 MRR milestone.
